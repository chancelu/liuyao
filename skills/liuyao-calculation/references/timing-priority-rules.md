# 应期优先级规则

## 目的

把答案卷里最常见、最容易混淆的应期断法，整理成一个优先级表。核心问题不是“能不能想到一个时间”，而是“多种时间线索同时出现时，先信哪一个”。

## 一、总顺序

遇到应期题，按这个顺序检查：

1. **先看卦期是否封死结果**
2. **再看用神是否旬空 / 伏藏 / 入墓 / 被合住**
3. **再看病因是克、泄、耗、还是原神受伤**
4. **再看动爻、变爻有没有给出直接触发点**
5. **最后才落到月日五行与具体时间**

如果前一层已经足够决定应期，不要跳到后一层乱报具体日子。

## 二、第一优先：卦期判断

### 1. 卦期内无望
若答案结构已经显示：
- 用神全局受伤严重
- 关键点不在本卦内显现
- 静卦且无解救路径
- 题目问的是“本月 / 年内 / 当前周期”

优先回答：
- 本卦周期内无望
- 本月不显
- 年内不显
- 出卦期后再看

### 2. 卦期内可成但拖
若结构显示可成，只是：
- 六合
- 勾陈
- 静中待动
- 财爻/父母爻休囚但未死

优先回答：
- 能成，但拖
- 当前不成，后续可成
- 应期不宜断得太近

## 三、第二优先：四类“锁住结果”的结构

### A. 旬空
优先级很高。
- 先看是否真空、假空、旺不为空
- 若题眼明显在“未落实”，先取出空/填实
- 若同时有生扶，写成：当前受空拖延，出空后转实

### B. 伏神
- 伏神不出，先看飞神是否压住
- 若飞神压伏，优先考虑“冲飞神”或“飞神失力”
- 只有冲飞神但伏神仍弱，不算真正成事

### C. 入墓
- 入墓先看是否需要冲墓、出墓
- 若墓库是题眼，优先考虑能冲开墓库的时间
- 若同时有旬空，先判断哪个更直接限制显化

### D. 合住
- 合住优先看“何时被冲开”
- 若合而成事，则不一定要破合；若合而锁住，则要以冲解
- 不能见合就一概说拖，要分“合成”还是“合绊”

## 四、第三优先：病因类型

### 1. 克用神
- 优先找制忌神、扶用神的时间

### 2. 泄用神
- 优先找能截断泄气链、并补用神的时间
- 答案卷里常见“木月木日 / 火月火日 / 水月水日”就是这种逻辑

### 3. 耗用神
- 看谁在耗，是否可被克制或转化

### 4. 原神受伤
- 若用神本身不死，但原神旬空/被克/被合住，优先救原神
- 很多“晚成”题，本质不是用神太差，而是原神暂时接不上

## 五、第四优先：动变直接给的触发点

### 1. 动爻
- 动爻若直接冲、合、化进、化退，优先级高于空泛五行类推
- 动爻形成“何时引发变化”的线索，常可直接落月日

### 2. 变爻
- 变爻若构成病因，优先以“解除变爻病因”为应期
- 变爻若构成结果，优先以“变爻坐实”为应期

### 3. 回头生 / 回头克 / 回头耗
- 出现回头作用时，应期常落在“回头作用被放大或被解除”的时间

## 六、第五优先：月日落点规则

当前面几层都明确后，再落时间：

### 1. 值日值月
- 用神、原神、关键动爻逢值，可作触发点

### 2. 冲开
- 适用于：伏神、入墓、合住、静而待动

### 3. 填实 / 出空
- 适用于：旬空、伏藏未显

### 4. 五行入局
- 不是单纯“这个五行旺”，而是要满足：
  - 生用神
  - 克病因
  - 泄/耗忌神
- 优先选能一举两得或三得的时间

## 七、常见冲突时怎么选

### 1. 旬空 vs 五行生扶
- 若未出空，再旺也可能只是假象
- 一般先处理空，再谈旺

### 2. 伏神 vs 冲飞神
- 冲飞神只是第一步
- 伏神得出且有力，才算真正转机

### 3. 入墓 vs 合住
- 先看哪一个更直接限制显化
- 若墓是核心，先冲墓；若合是核心，先破合

### 4. 月可成 vs 日可成
- 若题目所问带明显周期性（本月、今年、阶段性结果），优先落月
- 若是触发式小节点（拿通知、签字、找回物品），更容易落日

## 九、一句原则

**应期先判“锁”，再判“钥匙”，最后才报“时间”。**
