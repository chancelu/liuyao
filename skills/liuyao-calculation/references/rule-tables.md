# 规则表：装卦与分析速查

## 1. 六亲五行关系表

相对“我/本宫五行”来定：

- 生我者：父母
- 同我者：兄弟
- 我生者：子孙
- 我克者：妻财
- 克我者：官鬼

## 2. 八卦速查表

- 乾：天 / 金 / 西北
- 坤：地 / 土 / 西南
- 震：雷 / 木 / 东
- 巽：风 / 木 / 东南
- 坎：水 / 水 / 北
- 离：火 / 火 / 南
- 艮：山 / 土 / 东北
- 兑：泽 / 金 / 西

## 3. 六神顺序理解表

卷三强调六神依日干起例顺排。实际使用时先确认“起哪一神”，再按固定顺序排下去。

建议工作格式：
- 初爻：六神
- 二爻：六神
- 三爻：六神
- 四爻：六神
- 五爻：六神
- 上爻：六神

不要只记口诀，不落盘。

## 4. 常规分析顺序表

1. 题目核心
2. 世应
3. 用神
4. 原神/忌神/仇神
5. 月建
6. 日辰
7. 旬空
8. 动爻变爻
9. 进退飞伏
10. 结论

## 5. 常见问题取用神速查

- 问自己：世爻
- 问对方态度：应爻
- 问父母长辈：父母爻
- 问工作/官司/疾病/女测夫：官鬼爻
- 问兄弟朋友/竞争：兄弟爻
- 问财/资源/男测女：妻财爻
- 问子女/成果/医药/解除：子孙爻

## 6. 强弱判断最小清单

看用神时至少问六句：
- 得月建吗？
- 得日辰吗？
- 空吗？
- 破吗？
- 动吗？
- 有原神生扶还是被忌神压制？

## 7. 结果类型速查

- 成：用神得力，阻力可化
- 阻：用神弱且受制
- 延：用神不死但暂时无力
- 变：动变后结构改向
- 隐：伏藏、空亡、未显、不明
