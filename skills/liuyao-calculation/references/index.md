# 六爻计算索引

## 目的

把《云笈书院六爻卷》四卷内容整理成可执行、可检索、可复用的学习与计算框架。

## 四卷分工

- 卷一《易理基础卷》：阴阳、五行、天干、地支、基础宇宙模型
- 卷二《卦理基础卷》：河图、洛书、八卦、六十四卦、卦象与卦序
- 卷三《六爻基础卷》：起卦、装卦、世应、纳甲、六亲、六神、旬空/月建/日辰等基础参数
- 卷四《六爻用神卷》：用神、原神、忌神、仇神、进退、飞伏、双现、主辅用神
