# 基础框架：从易理到六爻

## 1. 学习分层

云笈四卷的逻辑很清楚：

1. 先懂世界模型：阴阳、五行、干支
2. 再懂卦的组织：河图、洛书、八卦、六十四卦
3. 再懂六爻的操作：起卦、装卦、参数
4. 最后懂判断核心：取用神与用神强弱、得失、受制情况

这意味着六爻计算不是“看到卦就断”，而是一个分层系统。

## 2. 卷一可直接转成的底层规则

### 阴阳
- 阴阳不是好坏，而是属性对待、消长、转化、对立统一。
- 六爻里最基础的符号就是阴爻与阳爻。
- 后续很多判断都依赖“动静、刚柔、内外、上下”的阴阳结构。

### 五行
- 五行是六爻里最基础的关系语言。
- 常见核心：生、克、泄、耗，以及旺衰背景。
- 六亲、用神、原神、忌神等关系，最终都要落回五行关系理解。

### 天干地支
- 地支在六爻中尤其关键，因为纳甲、旬空、月建、日辰都要用到。
- 卷一提供了理解地支时令、属性、对应系统的底层认知。

## 3. 卷二可直接转成的底层规则

### 河图、洛书
- 它们不是装饰性传统图案，而是后续方位、数理、象意、结构化记忆的来源。
- 实战里如果图难读，优先转成“方位 + 数 + 五行 + 阴阳”的文字表。

### 八卦
- 八卦是六十四卦的基础组件。
- 先记卦名、卦象、五行、方位、基本象意，再进入复卦层。

### 六十四卦
- 六爻判断时虽然重爻位与用神，但卦名、卦序、上下卦象意仍是背景层。
- 卦象适合做辅助理解，不宜替代装卦与用神判断。

## 4. 卷三与卷四在计算上的分工

### 卷三回答“盘面怎么建立”
- 怎么起卦
- 怎么定世应
- 怎么纳地支
- 怎么配六亲
- 怎么排六神
- 怎么看旬空、月建、日辰等参数

### 卷四回答“盘面里谁最重要”
- 问题对应哪一爻为用神
- 谁生它、谁克它、谁耗它
- 它是显还是伏，是进还是退，是单还是双现

## 5. 最小必备心法

- 不先装卦，不谈断卦。
- 不先定用神，不谈吉凶。
- 不脱离月建、日辰、旬空谈旺衰。
- 不因为某个象意很像，就跳过结构计算。
