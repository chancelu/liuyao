# 起卦与装卦

## 1. 起卦入口

### 摇卦起卦法
- 传统三枚铜钱，连续六次，自下而上记爻。
- 重点不是“软件怎么显示”，而是要知道每一爻如何从摇卦结果变来。
- 每爻都要记：阴/阳、动/静。

### 梅起爻断
卷三列出多种入口：
- 时间起卦法
- 报数起卦法
- 笔画起卦法
- 先后天八卦起卦法
- 按声音起卦法
- 丈尺寸起卦法
- 时加数起卦法
- 物数起卦法

适合没有铜钱、没有现成卦而又要快速立卦的场景。

## 2. 装卦总顺序

1. 得主卦
2. 定动爻，得变卦
3. 定世应
4. 纳地支
5. 配六亲
6. 配六神
7. 标月建、日辰、旬空等参数

## 3. 世应

世应是六爻里“我方 / 对方”“主体 / 客体”的关键锚点。

最实用理解：
- 世爻：我、我方、问题主体、自身立场
- 应爻：对方、对象、外部环境、对手、回应端

遇到软件盘面时，不要只抄结果；要知道世应是装卦结构的一部分。

## 4. 纳地支

纳甲/纳支是把六爻每个爻位接到地支系统上的关键步骤。

装卦时要形成这样的结果：
- 初爻：某支
- 二爻：某支
- 三爻：某支
- 四爻：某支
- 五爻：某支
- 上爻：某支

没有地支，就没法继续判断：
- 六亲
- 月建、日辰对各爻作用
- 旬空
- 合冲刑害类关系

## 5. 六亲配置

六亲不是按家庭关系死记，而是按“相对于本宫五行的生克关系”来定。

常用映射：
- 生我者：父母
- 同我者：兄弟
- 我生者：子孙
- 我克者：妻财
- 克我者：官鬼

所以六亲本质是五行关系标签，不只是人物标签。

## 6. 六神配置

六神用于增加事件色彩、表现方式、气质、征象层。

卷三强调：六神的排法依日干起例，按固定顺序顺排。

实务建议：
- 先把六神当作辅助层
- 不要用六神取代用神与旺衰判断

## 7. 装卦完成后的最小输出格式

建议固定写成：

- 本卦：X宫 X卦
- 变卦：X宫 X卦
- 动爻：第 X 爻
- 世爻：第 X 爻
- 应爻：第 Y 爻
- 六爻纳支：......
- 六亲：......
- 六神：......

这能防止后面分析时丢结构。
