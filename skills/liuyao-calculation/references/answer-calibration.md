# 题卷校准要点

## 目的

把《云笈书院六爻习题卷》答案卷里反复出现的断法，提炼成可复用的校准规则。这个文件不是替代基础规则，而是补足第一版 skill 在“题感、象法、应期”上的短板。

## 一、先分题型再作答

遇到题卷，不要一上来就综合断。先判断它属于哪类：

1. **取用神题**：只回答主用神/辅用神
2. **旺衰题**：只判断用神旺、相、休、囚、偏旺、偏弱，不抢先做综合吉凶
3. **象法题**：在旺衰结论之外，补事件细节、人物状态、拖拉/争吵/隐藏/被捡走等场景化信息
4. **应期题**：明确写出“病因 → 救应/触发 → 时间点”

## 二、答案卷的常见表达骨架

### 1. 旺衰卷常见骨架

- 先定：用神旺相 / 偏旺 / 偏弱 / 休囚
- 再补：日月、旬空、动爻、六冲六合、静卦/动卦
- 最后落结论：吉 / 凶 / 小凶转吉 / 凶而转吉

### 2. 象法卷常见骨架

- 先定用神状态
- 再用六神、六亲、动变解释人物/场景
- 再给“眼下如何、后面如何”
- 常见结尾：
  - 故此卦为吉
  - 故此卦不吉
  - 故此卦小凶转吉
  - 故此卦大凶

### 3. 应期卷常见骨架

必须显式写三步：

1. **病因在何处**
   - 旬空
   - 月破/日破
   - 飞神压伏神
   - 忌神克用神
   - 原神受伤
   - 某动爻泄耗太过
2. **何种力量能解**
   - 出空
   - 冲开飞神/墓库/合局
   - 某五行入局，克病因又生用神
   - 填实、值日、值月、逢合、逢冲
3. **落到时间**
   - 某月某日
   - 本卦周期内 / 本卦周期外
   - 短期无望 / 出月转机 / 年内不显

## 三、答案卷反复出现的判断词

### 1. 结果词

- 大吉
- 吉
- 偏吉
- 小吉
- 小吉转凶
- 小凶转吉
- 凶而转吉
- 凶
- 大凶
- 卦期内无望
- 本月不显 / 他月于卦不显

### 2. 过程词

- 拖拉
- 争吵
- 纠缠
- 被压下
- 伏藏不出
- 静而不动
- 冲散
- 回归状态
- 私下协商
- 朋友帮忙
- 价钱不合适
- 短期难成
- 先难后易 / 先凶后吉

## 四、答题时的约束

1. **只问旺衰时，不要越界写成综合命断**
2. **只问应期时，不能只写“某月某日”，必须交代为什么是那个时间**
3. **只问取用神时，优先回答主用神，必要时补辅用神，不展开太多**
4. **题卷语言可更像“批改答案”，实战咨询语言可更像“给用户解释”**
