# 象法补强清单

## 1. 使用方式

象法永远排在结构判断之后。

顺序固定为：
1. 先定用神与旺衰
2. 再看动静、世应、六神
3. 最后把它们翻译成事件细节

不要倒过来：不能先看一个六神就直接编故事。

## 2. 题卷里常见的象法落点

### 1) 拖拉 / 久拖不决
高频线索：
- 六合
- 勾陈
- 静卦不动
- 父母爻或财爻受困

常见表述：
- 不是短时间能成
- 会拖延
- 要回但不快

### 2) 争吵 / 口舌 / 沟通强
高频线索：
- 朱雀
- 官鬼或应爻强势
- 双方能量都旺

常见表述：
- 会有争吵
- 沟通频繁
- 会因为某事被训斥

### 3) 隐藏 / 私下 / 不明朗 / 被遮掩
高频线索：
- 玄武
- 伏神
- 不上卦

常见表述：
- 私下协商
- 信息不明
- 东西被藏/被掩盖
- 事情没摆到台面上

### 4) 刚硬 / 风险 / 受罚 / 病灾压力
高频线索：
- 白虎
- 官鬼旺
- 官鬼临世或关键位

常见表述：
- 风险大
- 压力重
- 会被罚
- 口舌官非加重

### 5) 曲折 / 虚惊 / 反复想法
高频线索：
- 腾蛇
- 游魂 / 归魂配合摇摆

常见表述：
- 自己想很多
- 心里没底
- 过程不顺

### 6) 熟人 / 协调 / 笨重推进
高频线索：
- 勾陈
- 兄弟爻、父母爻介入

常见表述：
- 中间人协调
- 事情推进慢
- 被现实事务牵制

### 7) 喜悦 / 帮助 / 顺手推进
高频线索：
- 青龙
- 原神生用神
- 子孙旺而无重克

常见表述：
- 有人帮忙
- 队友给力
- 后续走顺

## 3. 常见专题化落点

### 失物
- 玄武：藏、暗、被遮挡、私下拿走
- 勾陈：压着、埋着、卡在角落、近地
- 外卦：离得较远
- 内卦：离得较近
- 兄弟动：朋友帮找、他人介入、竞争夺物

### 感情/关系
- 应爻强：对方更强势
- 六合：关系拖着不断
- 六冲：散、变、冲开、很快见分晓
- 父母动：手续、证件、沟通文本

### 事业/官非
- 官鬼：规则、职位、压力、官方
- 父母：合同、通知、协议、流程
- 白虎+官鬼：处罚、重压、风险
- 朱雀+父母：文书口舌、通知消息
