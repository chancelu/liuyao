# 六爻计算工作流

## 1. 总流程

### A. 先问清楚输入类型

- 只有问题，没有卦
- 已有主卦和动爻
- 已有完整排盘
- 只有截图/图表

### B. 补齐缺口

#### 只有问题，没有卦
1. 选择起卦法
2. 得主卦与动爻
3. 得变卦
4. 装卦
5. 标参数
6. 取用神
7. 分析

#### 已有主卦和动爻
1. 先补装卦
2. 再标参数
3. 再取用神
4. 再分析

#### 已有完整排盘
直接做：
1. 校验盘面是否完整
2. 找用神
3. 看旺衰、动变、生克
4. 给结论

## 2. 每次都要显式检查的 8 个点

1. 问题主体是谁
2. 世应在哪里
3. 用神是谁
4. 原神、忌神、仇神是谁
5. 用神旺不旺
6. 用神动不动、变不变
7. 日月对用神如何
8. 结果是“成 / 阻 / 延 / 变 / 隐”中的哪一种

## 3. 断卦时的建议话术顺序

### 第一步：定位
- 这件事的主用神是……
- 辅助观察点是……

### 第二步：结构
- 用神得/失月建
- 得/失日辰
- 是否空、破、伏、受克
- 原神是否能生，用神是否被制

### 第三步：动态
- 动爻带来推进还是反转
- 变爻让事情变好还是变弱
- 是否有进神/退神
- 是否有飞伏关系

### 第四步：结论
- 当前更偏向：能成 / 难成 / 晚成 / 有条件可成 / 表面成而内里不稳
- 若能看时间，再单列“应期线索”

## 4. 图表类输入的处理

如果用户给的是排盘截图或模糊 PDF 图：

1. 先把图上信息拆成文本字段
2. 把每爻写成一行
3. 再开始判断

建议文本模板：

- 初爻：六亲 / 地支 / 六神 / 动静 / 是否世应
- 二爻：……
- 三爻：……
- 四爻：……
- 五爻：……
- 上爻：……

## 5. 最终输出模板

### 简版
- 问题核心：
- 用神：
- 关键结构：
- 判断：
- 备注：

### 详版
- 起卦/排盘依据
- 装卦结果
- 基础参数
- 用神系统
- 生克动变分析
- 结论
- 不确定点

## 6. 方法论原则

- 先结构，后象意
- 先主线，后细节
- 先用神，后神煞
- 先可验证规则，后经验修辞
