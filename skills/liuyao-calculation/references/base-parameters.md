# 基础参数

## 1. 为什么参数层不能跳过

卷三把“常规参数”单独成章，意思很明确：

同一用神，不同月建、日辰、旬空背景，判断会完全不同。

## 2. 最核心三项

### 月建
- 代表当月时令背景，偏长期、偏司权。
- 看爻得令与否、旺衰背景时，月建通常权重很高。

### 日辰
- 代表当日作用，偏即时、偏触发。
- 常用于看当下得失、冲合、生扶、日破等。

### 旬空
- 看某爻是否落空。
- “空”不等于永远没有，而是当前阶段力量、落实度、显化度不足。

## 4. 实战使用优先级

建议先看：
1. 月建
2. 日辰
3. 旬空
4. 动爻/变爻
5. 月破、日破、暗动
6. 再看伏吟、反吟、墓库、卦身、胎爻、间爻

不要一开始就被神煞带走。

## 5. 常用神煞

卷三另列：
- 天乙贵人
- 十干禄
- 驿马
- 羊刃
- 华盖
- 桃花
- 天赦

用法建议：
- 神煞做辅助提示，不作为核心立断依据。
- 若结构判断与神煞提示冲突，优先结构层：用神、动变、旺衰、日月。

## 6. 参数层的标准检查清单

在正式断卦前，逐项确认：

- 用神是否得月建
- 用神是否受日辰生扶或冲克
- 用神是否旬空
- 用神是否月破或日破
- 原神、忌神是否发动
- 动爻是否形成推进还是反转
- 是否有暗动、伏吟、反吟等强提示

## 7. 一句原则

参数层决定“这根爻有没有力、什么时候有力、当下有没有落地”。
