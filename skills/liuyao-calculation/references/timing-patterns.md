# 应期判定模板

## 1. 总原则

应期不要凭直觉报时间。先找“病因”，再找“解除病因或触发结果的时间”。

固定顺序：

1. 结果是否已成定局
2. 当前为什么没发生 / 为什么会发生
3. 什么力量能改变当前结构
4. 这种力量通常落在什么月日

## 2. 题卷答案中高频的病因类型

### A. 旬空
- 当前不落实、不显化、力不足
- 常见解法：出空、填实、值日值月
- 常见表达：
  - 用神旬空，出空则成
  - 财爻旬空，待出空后可回款

### B. 飞神压伏神
- 信息藏住、不出现、东西在但不露
- 常见解法：冲飞神、扶伏神、让伏神得出
- 常见表达：
  - 子日冲飞神午火，再生伏神子水
  - 飞神出空填实后，问题才解决

### C. 月破 / 日破 / 墓库 / 合住
- 爻受伤、被困、被锁、被合住
- 常见解法：冲开、出墓、破合、值旺
- 常见表达：
  - 父母爻入墓，待午月冲动
  - 合住文书，出月后金日可动

### D. 忌神太旺 / 原神受伤
- 用神本身不一定死，但补给断了或被压住
- 常见解法：让原神复位，或让忌神被制
- 常见表达：
  - 木月木日克土生火
  - 水月水日克火又生木

### E. 静卦不动 / 周期未到
- 事情暂时不会变
- 常见解法：等冲动、值旺、出本卦周期
- 常见表达：
  - 静卦主安静不动
  - 卦期内无望，出卦期才显

## 3. 常见“救应”模式

### 1) 一举两得 / 一举三得
优秀应期往往不是只扶用神，而是同时做到：
- 生用神
- 克病因
- 泄忌神 / 耗障碍

答题时可优先找这种时间点。

### 2) 冲而动
如果事情卡在：
- 伏藏
- 入墓
- 合住
- 静止

就优先考虑“冲”来引动。

### 3) 出空而实
如果题眼是“空”，重点不是玄断，而是：
- 什么时候出空
- 出空后有没有月日接力

### 4) 卦期内 vs 卦期外
如果答案卷多次出现“本卦周期内无望”，就说明当下结构根本不给结果。此时不要硬报一个月日。

## 4. 输出模板

- 当前病因在：
- 解除方式是：
- 因此应期更可能在：某月 / 某日 / 出空后 / 出月后 / 本卦周期外
- 置信度：高 / 中 / 低

## 5. 禁忌

- 不要只凭一个旺相爻就报应期
- 不要把“可能转机”写成“确定发生”
- 不要忽略题目问的是短期、月内、年内，还是泛问未来

## 6. 一句心法

**应期不是猜时间，而是给结构找开锁时点。**
