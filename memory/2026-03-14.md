### 关键教训
- `ctx.filter = 'blur()'` 在某些浏览器/环境不支持，会导致页面崩溃
- `mix-blend-mode: screen` 在 Canvas 上方的 HTML 元素上不生效
- AI 生成的图片质量难以满足高设计要求，纯 CSS/SVG 方案更可控
- 小北对设计非常挑剔，多轮否定是常态，要快速迭代不要恋战

### 三个关键规则确认（网上搜索验证）
1. **得月建定义**：月建生爻 或 同五行 → 旺相；月建与爻六合 → 也算旺相
2. **原忌仇是固定六亲表**：父母→原官鬼忌妻财仇子孙，以此类推（循环链）
3. **旺不为空**：得月建或得日辰（任一）即为旺，旬空不为空
4. **土的十二长生**：六爻中土"寄水宫"（长生在申，墓在辰），不是随火！卷三原文"辰为水土之墓库"佐证
