# 六爻网站《页面视觉与设计系统方向板 v0.1》

## 1. 设计目标

这个产品的视觉任务，不只是"做漂亮"，而是同时完成 4 件事：

1. 让用户一眼觉得高级
2. 让小白不害怕进入
3. 让结果显得可信、不是乱说
4. 让整个起卦过程像一次现代仪式体验

所以视觉方向不能做成：
- 工具网站
- 老式命理站
- 玄学贴图堆砌站
- 花里胡哨的新拟物 demo

它应该更像：
> 奢侈品官网的克制感 + 高级塔罗/星座网站的氛围感 + 现代产品的清晰交互

## 2. 品牌气质关键词

### 核心关键词
- 神秘、克制、现代、东方感、高级、仪式感、安静、深邃、可信、非工具化

### 次级关键词
- 细腻、有光泽、有留白、不急躁、有层次、有一点诗意

### 禁止气质
- 土豪玄学风、街头算命摊风、老黄历网站风、纯科技 dashboard 风、廉价 AI 产品风、赛博朋克过头风

## 3. 视觉世界观

### 3.1 世界观方向
建议不是"神棍神坛"，而是：
> 夜色、月相、墨色、玉石、金属微光、东方纹理、命运脉络

视觉灵感来源：
- 夜空与月光
- 墨与金
- 玉石和雾感
- 东方织物纹理
- 占卦中的"显现""推演""展开"

不是：
- 佛像、香炉、铜钱贴满屏、八卦阵硬铺背景

### 3.2 情绪基调

| 页面 | 情绪 |
|------|------|
| 首页 | 吸引、深邃、安静、想试一次 |
| 摇卦页 | 仪式感、专注、有参与感 |
| 结果页 | 稳、可信、看起来"有料"、不慌乱 |

## 4. 色彩系统

### 4.1 主背景色
深色系统为基底，不是纯黑，而是有温度、有层次的深色：
- `#0B0B0F` 深墨黑
- `#10111A` 蓝黑
- `#12131C` 炭黑
- `#16141D` 暗紫黑

### 4.2 辅助背景色
用于卡片和层级区分：
- `#171922` 深灰蓝
- `#1A1D28` 深墨紫
- `#202331` 半透明雾层

### 4.3 点缀色组合
**推荐组合：深墨黑 + 月白银 + 玉石青 + 少量暗金**

- 月白银：冷银、月白、浅金属色 → 高级、克制、奢侈感
- 玉石青：青绿偏灰、冷玉色 → 东方感、精致、不俗
- 暗金：低饱和金（不是亮黄/土豪金）→ 仪式感、品牌标识、关键按钮点睛

### 4.4 功能色
- 成功/积极：柔和玉青 / 松石绿（不要荧光绿）
- 警示/风险：暗琥珀 / 低饱和暖橙（不要亮红）
- 错误：深酒红 / 暗绯红（不要刺眼正红）

## 5. 字体与排版

### 5.1 字体组合
- 中文正文：思源黑体 / Noto Sans SC / HarmonyOS Sans
- 英文辅助：Inter / Geist / Manrope
- 标题：高级无衬线优先，不硬上古风字

### 5.2 排版原则
- 标题：允许更大字距、更松的呼吸感，奢侈品官网式大标题
- 正文：行高舒展、不挤、一屏内容不要太密
- 数据/排盘：层次清晰，用字号、透明度、间距区分信息层，不要像 Excel

## 6. 动效原则

### 6.1 总体原则
> 慢一点、轻一点、细一点，不炫技。

### 6.2 动效关键词
缓慢展开、淡入、微漂浮、光影扫过、显现、呼吸感

### 6.3 页面级动效
- 首页：背景轻流动感、光感/星点/纹理缓慢变化、Hero 文案优雅进入
- 摇卦页：铜钱/卦象交互有仪式感、每次摇卦有反馈、节奏要快不拖沓
- 生成页：像"推演展开"，不像 loading 圈圈
- 结果页：排盘区、结论区、详细分析按层出现，不一次性砸下来

### 6.4 动效时长
- Hover：150–220ms
- 卡片淡入：300–450ms
- 页面段落过渡：400–700ms
- 生成/推演类：800–1600ms

## 7. 组件风格

### 7.1 按钮
- 有重量感、精致，不做普通 SaaS 圆角实心按钮
- 主按钮：深底浅边或浅光泽边框，Hover 时轻微光泽变化
- 文案不要太功能化：开始起卦、摇这一爻、查看详细分析、保存这次结果

### 7.2 卡片
- 半透明深色卡片、轻边框、低对比阴影
- 有玻璃感但不要典型玻璃拟态 demo

### 7.3 输入框
- 较高内边距、深色底 + 柔和边框
- 焦点态有轻微发光或边线变化
- placeholder 文案有质感

### 7.4 Tooltip
- 深色浮层、边角精致、内容不要太长
- 适合解释术语，一两段为主

### 7.5 Tabs / 切换器
- segmented control 风格
- 细致、轻、低干扰

### 7.6 Progress / Step
- 不用粗暴进度条
- 轻量步骤点、脉络线、爻位进度视觉

## 8. 各页面视觉重点

### 8.1 首页
- 第一眼高级、有神秘氛围、不难懂
- Hero 大标题 + 抽象背景纹理 + 强 CTA + 示例问题区
- 不应出现：太多技术说明、太多术语、太密集内容

### 8.2 起卦问题页
- 安静、引导、不压迫
- 问题输入框成为页面核心
- 示例问题像灵感，不像 FAQ 列表

### 8.3 摇卦页
- 仪式感、参与感、一次次推进
- 中央交互非常强，6 次过程有"逐渐显现"的感觉
- 最适合做品牌记忆点的页面之一

### 8.4 生成页
- 显现、推演、稳定
- 不焦躁、不像系统在卡
- 像在进行一段严肃但优雅的推演

### 8.5 结果页
- 可信、有料、层次清晰
- 第一屏必须很能打（最容易截图和分享）
- 第一屏结构：问题信息 → 排盘总览（美观版）→ 初步结论卡 → CTA
- 平衡：不能太学术、不能太空洞、不能太工具、不能太玄

### 8.6 历史页
- 安静、收藏感、私人记录簿
- 像"私人卦例档案馆"，不是消息列表

### 8.7 分享图
- 可传播、高质感、一眼想保存
- 品牌视觉卡片，不是网页截图

## 9. 文案风格

### 9.1 总体文案气质
- 混合风格：首页有神秘高级氛围，过程对小白友好，深入内容有专业性

文案三层：
1. 第一层：短、轻、生活化、不吓人
2. 第二层：有一点诗意和氛围、但不要矫情
3. 第三层：专业、清楚、有逻辑、给爱好者信任感

### 9.2 文案禁忌
- 不要满屏"命运""天机""玄之又玄"
- 不要像街头算命先生
- 不要过于故作神秘
- 不要 AI 营销腔
- 不要"正在为您智能分析中请稍等"这种廉价感

## 10. 要避免的反例

### 10.1 视觉反例
1. 大红大金 → 廉价、旧站、民俗摊位感
2. 八卦图铺满背景 → 土、破坏高级感
3. 过度赛博玄学（紫蓝霓虹、粒子爆炸）→ 虚
4. SaaS 工具后台风 → 像表单系统，不像占卦体验

### 10.2 交互反例
1. 步骤过重（太多表单）→ 失去仪式感
2. 结果页像报告 PDF → 阅读压力太大
3. 分享图太像截图 → 传播欲望弱

## 11. 设计准则

### 11.1 设计准则
1. 所有页面优先考虑"氛围 + 可读性"
2. 深色系统为主，亮色只做点睛
3. 术语展示必须可解释，不要堆叠
4. 排盘必须有视觉设计，不可直接输出纯技术表格
5. 页面应有呼吸感，不可过密
6. 组件要像品牌化产品，不像后台模板
7. 动效要慢、稳、轻
8. 任何东方元素都要抽象化、现代化

### 11.2 前端实现准则
1. 不要用默认组件样式直接上线
2. 卡片、输入框、按钮要统一视觉系统
3. 动画统一 easing 和节奏
4. 所有 hover / active / loading 状态都要精细化
5. 结果页优先做成适合截图分享的布局
6. 首屏和摇卦页是重点打磨区

## 12. 设计参考方向

### 参考什么
- 奢侈品牌官网的留白和文字比例
- 高级美妆品牌官网的材质感
- 精品 tarot / astrology 网站的氛围与沉浸感
- Apple / 高级科技产品网站的节奏与动效克制

### 不要参考什么
- 国内老黄历网站
- 过度玄学营销页
- 廉价赛博占星页面
- 普通模板站

## 13. 一句话定义

> 做一个像高级数字占卜品牌，而不是算命工具网站。
> 神秘、克制、现代东方感，有仪式感，但交互必须清楚。
