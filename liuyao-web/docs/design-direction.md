# 六爻网站页面视觉与设计方向

## 1. 总体设计目标
这个产品的视觉不只是“做漂亮”，而是同时完成四件事：
- 让用户一眼觉得高级
- 让小白不害怕进入
- 让结果显得可信，不像乱说
- 让起卦过程像一次现代仪式体验

## 2. 品牌气质关键词
### 核心关键词
- 神秘
- 克制
- 现代
- 东方感
- 高级
- 仪式感
- 安静
- 深邃
- 可信
- 非工具化

### 禁止气质
- 土豪玄学风
- 街头算命摊风
- 老黄历 / 老旧周易站风
- 纯 SaaS dashboard 风
- 廉价 AI 产品风
- 赛博朋克过头风

## 3. 视觉世界观
建议灵感来源：
- 夜色与月光
- 墨色与雾感
- 玉石与金属微光
- 东方抽象纹理
- “显现”“推演”“展开”的感觉

不建议：
- 佛像
- 香炉
- 铜钱贴满屏
- 八卦阵硬铺背景

## 4. 页面设计关键词
全站统一：
- 神秘
- 克制
- 现代
- 东方感
- 高级
- 仪式感
- 留白
- 不土
- 不老旧
- 不工具化

## 5. 色彩系统建议
### 主背景
以深色系统为主：
- 深墨黑
- 蓝黑
- 炭黑
- 暗紫黑

参考方向：
- `#0B0B0F`
- `#10111A`
- `#12131C`
- `#16141D`

目标不是纯黑，而是有温度、有层次的深色。

### 辅助背景
用于卡片和层级区分：
- `#171922`
- `#1A1D28`
- `#202331`
- 半透明雾层

### 点缀色
最多 2～3 个，建议：
- 月白银
- 玉石青
- 少量低饱和暗金

推荐组合：
**深墨黑 + 月白银 + 玉石青 + 少量暗金**

## 6. 字体与排版
### 要求
- 高级感
- 中文可读性
- 现代感
- 不古早
- 不滥用书法字

### 建议
- 中文正文：思源黑体 / Noto Sans SC / HarmonyOS Sans
- 英文辅助：Inter / Geist / Manrope
- 标题允许更有质感，但没有合适中文 serif 时，宁可用高级无衬线

### 排版原则
- 标题可更大、更疏朗
- 正文行高舒展
- 一屏不要太密
- 排盘层次清楚
- 不要让排盘像 Excel

## 7. 动效原则
### 总原则
动效要：
- 慢一点
- 轻一点
- 细一点
- 不炫技

### 关键词
- 缓慢展开
- 淡入
- 微漂浮
- 光影扫过
- 显现
- 呼吸感

### 页面建议
- 首页：轻微流动背景 / 光感 / 星点
- 摇卦页：铜钱与卦象反馈动效要有仪式感
- 生成页：像推演展开，不像 loading 圈
- 结果页：排盘区、结论区、分析区分层显现

## 8. 组件风格
### 按钮
- 有重量感
- 精致
- 不像普通 SaaS 实心按钮
- hover 时有轻微光泽变化

按钮文案建议：
- 开始起卦
- 摇这一爻
- 查看详细分析
- 保存这次结果

### 卡片
- 半透明深色卡片
- 轻边框
- 低对比阴影
- 有玻璃感但不过度玻璃拟态

### 输入框
- 深色底
- 柔和边框
- 内边距舒展
- focus 态有轻微边线 / 光泽变化

### Tooltip
- 深色浮层
- 边角精致
- 一两段解释为主
- 作为术语解释与可信度组件

## 9. 页面视觉重点
### 首页
目标：第一眼高级、有神秘氛围、不难懂。

重点：
- Hero 大标题
- 抽象背景纹理
- 强 CTA
- 示例问题区

### 起卦页
目标：安静、引导、不压迫。

重点：
- 问题输入框成为页面核心
- 示例问题像灵感，不像 FAQ

### 摇卦页
目标：做出品牌记忆点。

重点：
- 中央交互主舞台
- 6 次摇卦逐步显现
- 有参与感，不像填表

### 生成页
目标：稳定、优雅、可信。

重点：
- 正在推演的感觉
- 不焦躁
- 不像系统卡住

### 结果页
目标：可信、有料、层次清晰。

**第一屏必须能打，且适合截图 / 分享。**

建议第一屏：
- 问题信息
- 排盘总览（美观版）
- 初步结论卡
- CTA

### 历史页
目标：像私人卦例档案馆，而不是普通列表。

### 分享图
目标：可传播、高质感、一眼想保存。

## 10. 文案风格
### 要做到三层并存
1. 短、轻、生活化
2. 有一点诗意和氛围，但不矫情
3. 专业、清楚、有逻辑

### 禁忌
- 不要满屏“命运”“天机”“玄之又玄”
- 不要像街头算命先生
- 不要过度故作神秘
- 不要 AI 营销腔
- 不要廉价系统提示语气

## 11. 必须避免的反例
### 视觉反例
- 大红大金
- 八卦图铺满背景
- 过度赛博玄学
- SaaS 后台风

### 交互反例
- 每一步都很重的表单
- 结果页像报告 PDF
- 分享图像网页截图

## 12. 实施准则
### 设计准则
- 优先“氛围 + 可读性”
- 深色为主，亮色只做点睛
- 术语必须可解释
- 排盘不可直接吐技术表格
- 页面要有呼吸感
- 东方元素必须抽象化、现代化

### 前端准则
- 不用默认组件样式直接上线
- 卡片、输入框、按钮统一视觉系统
- hover / active / loading 都要打磨
- 首屏、摇卦页、结果页优先做成品牌记忆点

## 13. 一句话方向定义
> 做一个像高级数字占卜品牌，而不是算命工具网站。
> 神秘、克制、现代东方感，有仪式感，但交互必须清楚。
