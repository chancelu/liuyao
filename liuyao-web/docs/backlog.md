# 六爻网站 MVP 开发 Backlog v0.1

## 优先级定义
- **P0**：必须做，不做就没有 MVP
- **P1**：强烈建议做，做了 MVP 会完整很多
- **P2**：后续优化项，不影响第一版主链路上线

---

## P0 Backlog（必须完成）

### P0-01 项目初始化与工程骨架 ✅
初始化 Next.js + TypeScript，接入 Tailwind CSS，建立项目目录结构，配置环境变量，配置 Vercel/Supabase 项目。

### P0-02 中文优先 + 多语言架构预埋 ✅
接入 i18n 方案，建立 zh-CN locale 文件，所有页面文案从 locale 文件读取，不在页面组件里写死中文文案。

### P0-03 数据库与认证基础设施 ✅
设计数据库 schema，建立 users/divinations/casts/charts/analyses/share_links 表，接入 Supabase Auth，实现邮箱注册/登录基础流程。

### P0-04 首页开发 ✅
Hero 区、三步说明区、示例问题引导区、轻内容说明区、CTA 区、导航与页脚基础版。

### P0-05 起卦问题页开发 ✅
问题输入框、分类选择、时间范围选择、背景补充输入、示例问题引导、表单校验、提交创建 divination 记录。

### P0-06 摇卦交互页开发 ✅
摇卦状态管理、三枚铜钱交互组件、第 N/6 次进度显示、每次爻结果记录、重新开始功能、完成后提交 cast 数据。

### P0-07 排盘引擎开发 ✅
定义六爻原始输入格式，实现主卦/变卦生成，实现动爻识别，实现世应、纳甲、六亲、六神、月建/日辰/旬空，输出标准化 chart JSON。

### P0-08 排盘生成过渡页开发 ✅
展示问题摘要、主卦/变卦概览、推演状态文案，对接排盘与分析异步流程。

### P0-09 AI 分析服务开发
定义 analysis input/output schema，建立 analysis service，对接大模型调用，接入 liuyao-calculation 分析链路，生成初步结论/白话分析/专业分析，保存分析结果。

### P0-10 结果页开发 ✅（基础版）
顶部摘要区、排盘总览区、初步结论卡、白话分析模块、专业分析折叠模块、结果数据渲染、保存/分享/重新起卦按钮。

### P0-11 游客 1 次体验机制 ✅
游客 session 机制、游客体验次数记录、游客最多体验 1 次限制、游客结果临时保存、引导注册后绑定账户。

### P0-12 注册后保存结果 ✅
结果页"保存结果"入口、未登录时跳转/弹出注册、注册成功后绑定当前 divination、保存成功提示。

## P1 Backlog（强烈建议首版做）

### P1-01 历史记录页 ✅（基础版）
卦例列表、卡片样式、点击查看详情、再次起卦入口、空状态设计。

### P1-02 Tooltip 术语解释
建立术语词典、结果页术语高亮、tooltip 交互、中文解释内容。

### P1-03 示例问题引导优化
首页示例问题、起卦页示例问题、输入空状态引导、分类联动推荐问题（可选）。

### P1-04 重新起卦功能 ✅
结果页重新起卦按钮、历史页再次起卦入口。

### P1-05 分享链接 ✅
创建 share token、生成公开分享链接、开发 share 页面、控制分享内容字段。

### P1-06 分享图片
定义分享图布局、生成图片卡片、提供下载/复制功能、支持中文文案渲染。

### P1-07 摇卦页体验打磨 ← 当前
铜钱动画优化、过渡效果优化、进度反馈优化。

### P1-08 结果页视觉打磨 ← 当前
排盘视觉设计升级、初步结论卡 refinement、白话/专业层级优化、按钮和模块层次打磨。

### P1-09 埋点系统
首页/提问页/摇卦页/结果页/注册/保存/分享埋点。

### P1-10 基础后台报表页
漏斗报表、分类分布、注册转化、分享率、分析失败率。

## P2 Backlog（后续优化）

### P2-01 高保真首页品牌打磨
更完整的动态背景、更成熟的品牌文案、更强的奢侈感/现代东方感细节。

### P2-02 更细的分享策略
可选匿名问题、多种分享图模板、社交平台预览优化。

### P2-03 更丰富的结果组织
结果摘要标签体系、结果推荐动作、更漂亮的趋势模块。

### P2-04 分析质量回溯工具
查看某次分析原始输入、查看模型输出、查看 skill 版本。

### P2-05 用户反馈机制
对结果有帮助吗、简单反馈入口、错误案例收集。

### P2-06 内容模块（轻）
六爻怎么问更清楚、新手引导内容、常见问题说明。

### P2-07 多语言真正上线
增加英文 locale、分享图多语言适配、文案与 SEO 多语言版本。
