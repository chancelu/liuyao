## 7. 旬空（出处：卷三·第十一课 + 增删卜易）
甲子旬中戌亥空、甲戌旬中申酉空、甲申旬中午未空
甲午旬中辰巳空、甲辰旬中寅卯空、甲寅旬中子丑空

### 旬空判断规则（增删卜易·野鹤曰）
不为空（假空，出空即有用）：
- 旺不为空：爻得月建或日辰生扶（同五行或被生），即为旺相，虽旬空亦不为空
- 动不为空：动爻虽旬空，亦不为空
- 有日建动爻生扶者，亦不为空
- 动而化空：动爻变出之爻旬空（需注意，有争议）
- 伏而旺相，皆不为空

为空（真空或实空）：
- 月破为空
- 有气不动亦为空（有气但静爻且旬空）
- 伏而被克亦为空
- 真空为空：春土、夏金、秋木、冬火（即当季最弱的五行逢旬空 = 真空，无救）

### 代码判断简化
```
isXunkong(branch, dayStem) → boolean  // 是否落在旬空范围
isReallyEmpty(branch, monthBranch, dayBranch, isMoving, ...) → boolean
  if !isXunkong → false (不空)
  if isMoving → false (动不为空)
  if 得月建或得日辰 → false (旺不为空)
  if 真空(季节最弱五行) → true (真空)
  if 月破 → true
  else → true (有气不动为空)
```

## 8. 月破（出处：卷三·第十一课）
月建与爻相冲为破。即月建地支与爻地支构成六冲关系。

## 9. 日破（出处：卷三·第十一课）
用神或其他爻在休囚或逢空不动的状态下，与日辰相冲。
前提：休囚 + 逢空/不动 + 与日辰相冲

## 10. 暗动（出处：卷三·第十一课）
旺相之静爻，与日辰对冲，冲则动。
前提：旺相 + 静爻 + 与日辰相冲
分析时与动爻通论。

## 11. 进神与退神（出处：卷四·第三课 + 卷三·第四课）

### 进神（动而化进，同五行地支前进）
寅化卯、巳化午、申化酉、亥化子、丑化辰、辰化未、未化戌、戌化丑

### 退神（动而化退，同五行地支后退）
卯化寅、午化巳、酉化申、子化亥、辰化丑、丑化戌、戌化未、未化辰

## 12. 六亲配置（出处：卷三·第九课）
以卦宫五行为"我"：
- 生我者 → 父母
- 比和者（同我） → 兄弟
- 我生者 → 子孙
- 我克者 → 妻财
- 克我者 → 官鬼

变卦六亲：地支以变卦纳支，但"我"仍以主卦宫五行为基准。

## 13. 六神配置（出处：卷三·第十课）
以摇卦当天日干为准，从初爻起配：
甲乙起青龙、丙丁起朱雀、戊日起勾陈、己日起螣蛇、庚辛起白虎、壬癸起玄武

顺序：青龙→朱雀→勾陈→螣蛇→白虎→玄武（循环）

## 14. 取用神规则（出处：卷四·第一课至第七课）

### 基本取用
- 自占/自身之事 → 世爻
- 对方/他人/对手 → 应爻
- 父母长辈/房屋车船/文书合同/工作单位 → 父母爻
- 工作职位/官司/女测夫/疾病忧患 → 官鬼爻
- 兄弟同辈/竞争者 → 兄弟爻
- 钱财资源/男测女 → 妻财爻
- 子女/医药解除/福神 → 子孙爻

### 按问题类别映射
- 问感情（男问女）→ 妻财爻
- 问感情（女问男）→ 官鬼爻
- 问事业/工作 → 官鬼爻（辅看父母爻=文书/合同）
- 问财运 → 妻财爻
- 问健康/疾病 → 官鬼爻（辅看子孙爻=医药、世爻=自身）
- 问学业/考试 → 父母爻（以录取通知书/文书为准）
- 问失物 → 妻财爻
- 问出行 → 世爻
- 问天气 → 动爻（不知何天气时）；父母爻（问雨）
- 问比赛 → 世爻（兼看应爻）
- 问升职 → 父母爻（以委任书为准，官鬼辅助）
- 问签约/合同 → 父母爻
- 问官司 → 官鬼爻（兼看世应）
- 六亲之事 → 以对应六亲为用（不管问什么事）
- 不在六亲之类 → 取应爻

### 用神双现取用优先级（出处：卷四·第五课）
动爻（两爻均动则旺相优先）> 世爻 > 应爻 > 离世爻近（距离相同则旺相优先）

额外规则（从百题库答案提取）：
- 旬空的爻优先级低于不空的爻
- 日破/月破的爻优先级低于不破的爻
- 暗动的爻可视为动爻
- 持世且动爻 > 单纯动爻
- 应爻 > 离世爻近

### 主辅用神（出处：卷四·第六课）
- 主用神：一卦中分析事情的核心取用爻
- 辅用神：一卦中分析事情的辅助取用爻
- 例：男问女感情 → 主用神=妻财爻，辅用神=兄弟爻(竞争者)、世爻(自身)、应爻(事情本身)

## 15. 伏神与飞神（出处：卷四·第四课）

### 伏神
卦中六亲不全，用神不上卦时，从所在宫的首卦（八纯卦）中借用。

### 伏神有用条件
1. 伏神得日月生
2. 伏神旺相，合日辰或合月建
3. 伏神得飞神生
4. 伏神得动爻生
5. 日月动爻冲克飞神，飞神被冲开
6. 飞神空破休囚入库，无力克害伏神

### 伏神无用条件
1. 伏神休囚无气
2. 伏神被日月冲克
3. 伏神被旺相之飞神克害
4. 伏神墓绝于日月飞神
5. 伏神休囚旬空

## 16. 用神旺衰判断（综合卷一·卷三 + 增删卜易 + 网络确认）

### 得月建（月令旺衰）— 分层次
月建地支五行与爻地支五行的关系：

高层次旺相：
- 爻与月建同五行 → 临月建（最旺）
- 爻与月建六合 → 得月合（旺）

一般层次旺相：
- 月建生爻 → 得月生（相）
- 爻与月建同五行但非临建 → 得月令扶（旺）

衰弱：
- 月建克爻 → 受月克（衰）
- 爻克月建 → 耗力（不旺不衰，偏弱）
- 爻生月建 → 泄气（不旺不衰，偏弱）

简化判断（代码用）：
- 月建生爻 或 月建与爻同五行 → 得月建 = 旺相
- 月建克爻 → 受月克 = 休囚
- 其余 → 不得不失

### 得日辰
同上逻辑，用日辰地支五行判断。日辰为即时力量，与月建同权同功。

### 旺衰综合判断
1. 得月建 + 得日辰 → 旺相（最强）
2. 得月建 + 不得日辰 → 偏旺
3. 不得月建 + 得日辰 → 偏旺
4. 不得月建 + 不得日辰 → 休囚
5. 月建或日辰克爻 → 受克（衰）

### 强弱最小清单
- 得月建吗？（同五行或月生）
- 得日辰吗？（同五行或日生）
- 空吗？
- 破吗？
- 动吗？
- 有原神生扶还是被忌神压制？

## 17. 原神、忌神、仇神（出处：卷四·第二课 + 网络确认）

### 定义
- 原神：生用神的六亲（用神动力之根源）
- 忌神：克用神的六亲（用神所畏忌）
- 仇神：克原神/生忌神的六亲（间接削弱用神）

### 六亲固定对应表（已确认）
| 用神 | 原神 | 忌神 | 仇神 |
|------|------|------|------|
| 父母 | 官鬼 | 妻财 | 子孙 |
| 官鬼 | 妻财 | 子孙 | 兄弟 |
| 妻财 | 子孙 | 兄弟 | 父母 |
| 子孙 | 兄弟 | 父母 | 官鬼 |
| 兄弟 | 父母 | 官鬼 | 妻财 |

说明：这是固定表，因为六亲本身就编码了五行关系。
例：父母为用神 → 官鬼生父母（克我者生生我者）→ 官鬼为原神
    妻财克父母（我克者克生我者）→ 妻财为忌神
    子孙生妻财且克官鬼 → 子孙为仇神

### 五行生克链推导
生克链：兄弟 → 子孙 → 妻财 → 官鬼 → 父母 → 兄弟（循环）
- 用神的前一位 = 原神（生用神）
- 用神的后两位 = 忌神（克用神）
- 忌神的前一位 = 仇神（生忌神 = 克原神）

## 20. 世应配置（出处：卷三·第七课）

### 歌诀
八卦之首世六当，以下初爻轮上扬，游魂七卦四爻立，归魂八卦三爻详。

### 规则
- 八纯卦（宫首卦）：世在六爻（上爻），应在三爻
- 一世卦：世在初爻，应在四爻
- 二世卦：世在二爻，应在五爻
- 三世卦：世在三爻，应在上爻
- 四世卦：世在四爻，应在初爻
- 五世卦：世在五爻，应在二爻
- 游魂卦（第七卦）：世在四爻，应在初爻
- 归魂卦（第八卦）：世在三爻，应在上爻

### 应爻定位
世爻定下来，往下隔两爻就是应爻的位置。
